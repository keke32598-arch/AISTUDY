# AI API 调用

## 什么是AI API

AI API是大模型提供商（如OpenAI、Claude、Anthropic等）提供的接口，允许开发者通过HTTP请求与AI模型进行交互，获取模型的生成结果。

## 主流AI API提供商

| 提供商 | 主要模型 | API文档 | 特点 |
|-------|---------|--------|------|
| OpenAI | GPT-4, GPT-3.5 | https://platform.openai.com/docs | 功能全面，生态成熟 |
| Anthropic | Claude 3 | https://docs.anthropic.com | 长文本处理能力强 |
| Google | Gemini | https://ai.google.dev | 多模态能力出色 |
| 百度 | 文心一言 | https://cloud.baidu.com/doc/WENXINWORKSHOP/s/3l5qkqg1m | 中文支持好 |
| 阿里 | 通义千问 | https://help.aliyun.com/zh/dashscope/developer-reference/api-details | 中文支持好 |
| 腾讯 | 混元大模型 | https://cloud.tencent.com/document/product/1729 | 中文支持好 |

## API调用的基本流程

1. **获取API密钥**：在提供商平台注册并获取API Key
2. **安装SDK**：安装官方提供的SDK或使用HTTP客户端
3. **配置认证**：设置API Key和其他必要参数
4. **构建请求**：准备输入参数（如模型选择、提示词等）
5. **发送请求**：调用API并等待响应
6. **处理响应**：解析返回的结果

## OpenAI API调用示例

### 基本聊天完成

```python
import openai

# 配置API Key
client = openai.OpenAI(api_key="your-api-key")

# 发送请求
response = client.chat.completions.create(
    model="gpt-3.5-turbo",
    messages=[
        {"role": "system", "content": "你是一个乐于助人的助手。"},
        {"role": "user", "content": "你好，如何学习Python？"}
    ]
)

# 处理响应
print(response.choices[0].message.content)
```

### 流式输出

```python
import openai

client = openai.OpenAI(api_key="your-api-key")

# 流式调用
stream = client.chat.completions.create(
    model="gpt-3.5-turbo",
    messages=[
        {"role": "user", "content": "写一段关于人工智能的短文"}
    ],
    stream=True
)

# 逐字输出
for chunk in stream:
    if chunk.choices[0].delta.content:
        print(chunk.choices[0].delta.content, end="")
```

## API调用的最佳实践

### 1. 错误处理

```python
import openai
from openai import OpenAIError

try:
    # API调用代码
    pass
except OpenAIError as e:
    print(f"API错误: {e}")
except Exception as e:
    print(f"其他错误: {e}")
```

### 2. 速率限制处理

```python
import time
import openai

def call_with_retry(prompt, max_retries=3):
    for i in range(max_retries):
        try:
            response = openai.OpenAI().chat.completions.create(
                model="gpt-3.5-turbo",
                messages=[{"role": "user", "content": prompt}]
            )
            return response
        except openai.RateLimitError:
            print(f"速率限制，等待后重试...")
            time.sleep(2 ** i)  # 指数退避
    raise Exception("达到最大重试次数")
```

### 3. 成本控制

- **合理设置max_tokens**：避免生成过长的内容
- **使用缓存**：对相同的请求结果进行缓存
- **批量处理**：将多个小请求合并为一个大请求
- **监控使用量**：定期检查API使用情况和费用

## 安全注意事项

1. **API Key保护**：不要将API Key硬编码在代码中，使用环境变量或配置文件
2. **输入验证**：对用户输入进行验证，防止Prompt注入攻击
3. **输出过滤**：对模型输出进行过滤，防止生成有害内容
4. **数据隐私**：不要发送敏感信息给API

## 实践项目

### 简单聊天机器人

```python
import openai

class ChatBot:
    def __init__(self, api_key, model="gpt-3.5-turbo"):
        self.client = openai.OpenAI(api_key=api_key)
        self.model = model
        self.messages = []
    
    def add_message(self, role, content):
        self.messages.append({"role": role, "content": content})
    
    def chat(self, user_input):
        self.add_message("user", user_input)
        
        response = self.client.chat.completions.create(
            model=self.model,
            messages=self.messages
        )
        
        assistant_response = response.choices[0].message.content
        self.add_message("assistant", assistant_response)
        
        return assistant_response

# 使用示例
if __name__ == "__main__":
    bot = ChatBot(api_key="your-api-key")
    
    while True:
        user_input = input("你: ")
        if user_input.lower() == "退出":
            break
        response = bot.chat(user_input)
        print(f"机器人: {response}")
```

## 常见问题

1. **API Key无效**：检查API Key是否正确，是否有使用权限
2. **速率限制**：API调用频率过高，需要等待或申请更高的限额
3. **模型不存在**：检查模型名称是否正确
4. **参数错误**：检查请求参数是否符合API文档要求
5. **网络问题**：检查网络连接，考虑使用代理
