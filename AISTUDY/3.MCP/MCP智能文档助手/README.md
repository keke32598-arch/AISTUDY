# MCP智能文档助手

## 项目简介

MCP智能文档助手是一个基于第三方MCP服务器的小型工具，用于读取和分析文档文件，提取关键信息，生成智能摘要，并提供基于文档内容的问答功能。

## 功能特点

- 使用第三方MCP服务器读取文档文件
- 支持常见文档格式：TXT、MD、CSV
- 使用通义千问AI模型生成文档摘要
- 基于文档内容提供智能问答功能
- 当MCP服务器不可用时自动切换到本地模式
- 提供友好的命令行交互界面
- 支持文件内容预览，确保读取正确的文件

## 技术栈

- Python 3.7+
- DashScope SDK（通义千问API）
- requests（HTTP请求）
- MCP协议（Model Context Protocol）

## 安装依赖

```bash
pip install dashscope requests
```

## 配置

1. 在 `config.py` 文件中设置你的通义千问API Key：

```python
API_KEY = "your-api-key"  # 请替换为你的通义千问API密钥
```

1. 配置MCP服务器地址（如果使用**第三方MCP服务**器）：

```python
MCP_SERVER_URL = "http://localhost:8000/mcp"  # MCP服务器地址
```

## 工作原理

1. **用户输入**：用户通过命令行界面输入操作命令和参数
2. **文件读取**：通过MCP客户端调用第三方MCP服务器的文件读取工具，或在本地模式下直接读取文件
3. **文档分析**：分析文档内容，提取关键词和结构信息
4. **AI处理**：调用通义千问API生成摘要或回答问题
5. **结果展示**：将处理结果展示给用户

## 注意事项

- 需要有效的通义千问API Key才能使用AI相关功能
- 如果MCP服务器不可用，会自动切换到本地模式
- 支持的文件格式：TXT、MD、CSV
- 大文件可能需要较长的处理时间

## 故障排除

### MCP服务器不可用

如果MCP服务器不可用，程序会自动切换到本地模式，直接读取本地文件。

### API调用失败

- 检查API Key是否正确
- 检查网络连接是否正常
- 检查API调用频率是否超过限制

### 文件读取失败

- 检查文件路径是否正确
- 检查文件是否存在
- 检查文件权限是否正确
